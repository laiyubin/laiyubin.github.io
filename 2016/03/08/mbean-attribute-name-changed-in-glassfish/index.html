MBean attribute name changed in Glassfish

I am studying MBean following Spring’s document.
Srping Framework 2.5 -> Chapter 20. JMX

I created a servlet for retrieve attribute value from MBean, then I deployed my project to Glassfish 4.1.1.

But I found I cannot get attribute(name, age) value, so I add some statements to print out the actually MBean.

Seems not problems. wait, why the first character is capital?

So I use Age, Name as the attribute name, finally, everything is OK now.
But I still don’t know why the attributes name changed in Glassfish server.
